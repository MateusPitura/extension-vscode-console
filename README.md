## Description

Easily insert `console.log` or equivalent print statements into your code across multiple programming languages

A [Chrome](https://github.com/MateusPitura/extension-chrome-console) extension is available for optional use to enhance productivity

## How to Run

1. In VSCode search for `Star Console` in extensions tab

2. Select some code and press `Shift` + `Alt` + `L`. This will insert a print statement in the file's programming language, beginning with the emoji üå†. It also supports empty selections

**For devs:**

1. Install packages with `npm i`

2. To test, press `F5`

3. Update the CHANGELOG

4. Update `version` references and run `npm i`

5. Compile with `npx vsce package`

6. Upload the `.vsix` to Visual Studio Marketplace

7. Create a tag in Git and upload to GitHub

8. Create a new release on GitHub
